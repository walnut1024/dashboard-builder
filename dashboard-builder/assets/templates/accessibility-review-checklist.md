# Accessibility Review Checklist

- Text contrast is adequate for the target display.
- Non-text marks and status indicators have sufficient contrast.
- Color is paired with labels, shape, ordering, line style, or position.
- Essential data is not available only through hover.
- Motion has reduced-motion or static fallback.
- Alerts do not rely on color alone.
- Keyboard/focus paths exist for interactive production dashboards.
- Touch targets are usable when narrow or touch fallback is required.
- Screenshot/export preserves labels, units, caveats, and source status.
